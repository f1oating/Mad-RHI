# Mad-RHI

A lightweight Rendering Hardware Interface (RHI) abstraction layer.

## Examples

<details>
<summary>Triangle</summary>

![Triangle](docs/screenshots/triangle.png)

A minimal hello-world example. Renders colored triangle using a hardcoded vertex buffer and a basic vertex/fragment shader pipeline. Color changes with delta time.

</details>

<details>
<summary>Exponential Fog</summary>

![Exponential Fog](docs/screenshots/fog_comparison.png)

Demonstrates depth-based exponential fog applied to a scene of cubes.

</details>

<details>
<summary>God Rays</summary>

![GodRays](docs/screenshots/godrays.jpg)

Volumetric light scattering via ray marching through a shadow-mapped scene using the Henyey-Greenstein phase function. Rendered at half resolution and composited over the color pass.

</details>
